[ngSwitch]

Se utiliza la propiedad [ngSwitch], para identificar el día ingresado, y se utiliza la propiedad *ngIf para indicar que si el usuario ingresa 0, no se muestre ningún texto

Lunes Martes Miercoles Jueves Viernes Sabado Domingo No valido

[ngSwitch]='dia' *ngIf="dia". *ngSwitchCase='1'. *ngSwitchDefault
